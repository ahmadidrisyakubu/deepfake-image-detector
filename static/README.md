# Static Assets

## Logo.jpg

Please replace the placeholder `Logo.jpg` file with your actual logo image.

The logo should be:
- Format: JPG or PNG
- Recommended size: 80px height (width auto)
- Used in the header of the application

If you don't have a logo, you can:
1. Use a free logo from [Flaticon](https://www.flaticon.com)
2. Generate one using AI tools
3. Remove the logo reference from `templates/index.html` (line 9)
